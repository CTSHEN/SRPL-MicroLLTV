This file is the start up procedire of ros nodes for vertical system 
component test.

1. Start the state esitmation system

$ roslaunch srpl_micro_lltv state_estimate.launch

2. Start a ros bag to record all topic data

$ rosbag record -a

3. Start the valve control and heat up the thruster

$ rosrun srpl_micro_lltv thruster_control_gpio.py

4. Start the component test

$ rosrun srpl_micro_lltv system_vtest.py
