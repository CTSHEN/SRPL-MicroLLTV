This file is the start up procedire of ros nodes for vertical system 
component test.

1. Start the state esitmation system

$ roslaunch srpl-micro-lltv state_estimate.launch

2. Start a ros bag to record all topic data

$ rosbag record -a

3. Start and test the purge valve

$ rosrun srpl-micro-lltv purge_valve.py

4. Start the valve control and heat up the thruster

$ rosrun srpl-micro-lltv thruster_control_gpio.py

5. Start the component test

$ rosrun srpl-micro-lltv system_vtest.py
